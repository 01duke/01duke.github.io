Visual Hierarchy

What is Visual Hierarchy?

Visual Hierarchy is the idea of presenting an object to be more important than other objects. Using the right colors in the right places. Using a bright color to draw attention to important objects on the page like the yellow search jobs button on the U.S. Army's website. spacing can help give an object a of importance. giving an object more space can draw more eyes to, the search jobs button is given its own space drawing the viewers eyes in.
